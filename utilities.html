Utilitiy functions for dealing with WEAVE data

Parsing codes in the FITS headers

The header entry “OBSTEMP” encodes the requested observational constraints for the OB: the seeing, transparency, elevation, lunar angle and sky brightness. The OBSTEMP code consists of a letter for each constraint. This converts the codes to numerical values and adds airmass.

Convert an OBSTEMP code to a dictionary of constraint values.

Type Details
codes DataArray a DataArray of OBSTEMP codes
Returns dict the numerical value for each observing constraint

Parsing line names

Line names in the galaxy table are a combination of the species and restframe wavelength, e.g. [OIII]_5006.77. This splits them and returns the species and wavelength separately.

Type Details

Returns tuple the species and wavelength of each line

ks_norm_prob

 ks_norm_prob (x:xarray.core.dataarray.DataArray, dim='SPEC', cdf=<bound
               method rv_continuous.cdf of
               <scipy.stats._continuous_distns.norm_gen object at
               0x7f1a31383730>>)

Calculate the KS probability that x are drawn from a standard Normal distribution.

The implementation is based on scipy.stats.ks_1samp and adapted to xarray for speed.

The input DataArray must be 2D, with one dimension to be kept (specified by dim) and the other for which the distribution of values will be tested. A multi-dimensional DataArray may be converted to 2D using DataArray.stack.

Type Default Details
x DataArray a 2D array of values to test
dim str SPEC the dimension of x to retain
cdf method cdf the cdf to compare against
Returns DataArray

add_expid

 add_expid (ds:xarray.core.dataset.Dataset)

Add EXPID coord to a Dataset.

This is constructed by numbering filename/RUNs for each CAMERA in order of MJD and adding this to OBID * 100000000, to create a unique ID for each exposure (i.e. a pair of RUNs taken with each CAMERA).

Type Details
